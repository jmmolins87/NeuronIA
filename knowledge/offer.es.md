# ClinvetIA - Oferta

## Que es ClinvetIA
- ClinvetIA es una agencia de marketing digital para clinicas veterinarias pequenas (una sede) que usa agentes de IA para mejorar la atencion y convertir interes en citas.
- Objetivo: responder mas rapido, captar mejor la demanda y reducir perdidas por respuestas tardias.

## Para quien es
- Clinicas veterinarias de una sede con recepcion ocupada.
- Equipos pequenos que no pueden estar pendientes 24/7.
- Clinicas que quieren un proceso claro para convertir preguntas en agenda (sin humo).

## Problemas que ayuda a resolver
- Respuestas lentas fuera de horario.
- Demasiadas conversaciones repetitivas (precios orientativos, horarios, ubicacion, servicios).
- Dificultad para filtrar urgencias vs dudas generales.
- Poca continuidad entre "pregunta" -> "reserva".

## Lo que NO prometemos
- No garantizamos resultados, ingresos, ROI, ni numero de citas.
- No afirmamos tener casos reales o testimonios (empresa nueva).
- No damos consejo medico veterinario.

## Que incluye una demo
- Revision rapida de tu caso (tipo de clinica, volumen, canales).
- Explicacion del flujo: web -> asistente -> reserva.
- Recomendacion de siguiente paso (si encaja o no).

## Proxima fase (solo si el usuario pregunta)
- Se puede explorar integracion con WhatsApp/email/CRM como fase posterior.
- No se debe presentar como algo ya activo si no esta implementado.
