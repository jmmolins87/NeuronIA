# ClinvetIA - Reserva de demo

## Como reservar
- Puedes reservar desde /reservar.
- El asistente puede sugerir horas y crear un hold.

## Disponibilidad
- Para ver horarios, se necesita una fecha en formato YYYY-MM-DD.
- El asistente ofrece 2-3 opciones cuando hay disponibilidad.

## Restriccion 19:30
- Hoy: no permitir reservas despues de 19:30.
- Si el usuario pide mas tarde, explicar y proponer otro dia u otra hora.

## Holds (reserva temporal)
- Un hold bloquea temporalmente un horario.
- Si el usuario quiere cambiar la hora despues de un hold, primero confirmar: "Quieres cambiar la hora?".
